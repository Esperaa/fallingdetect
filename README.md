- #### Version 2.0
###### 第二代设备的改进主要包括：

- ###### 在电路板的背面增加了一层保护塑料垫，以防止冲击并保护后面的电路。
- ###### 已取消版本1.0的初始放置位置；检测将基于开机后的放置位置（从而提高了可用性）。
- ###### 取消了发送警报通知的延迟，允许立即发送警报信息。取消按钮现在仅用于静音蜂鸣器警报和重新启动程序。
